FAQS
How do I use a pedal to trigger an event?
If your keyboard has a spare pedal input, it's probably possible to use it directly. Plug the pedal in, open the Input Monitor (Control | Show Input Monitor), see what kinds of signals it's sending, and set up a global trigger to work with those. If you're out of pedal inputs or can't set the keyboard to send a usable signal, you can use a pedal box from MIDI Solutions to cause the pedal to trigger any note. A common thing to do is to set it to send a note beyond the range of your keyboard (for example, C#8). Even if Cadenza isn't set to ever play anything with that note, it will still receive the MIDI message and can work with it.
How do I get Cadenza to use a synthesizer that isn't listed?
Cadenza uses plain text files to define synthesizers, their built-in and expansion patch banks, and the commands necessary to switch patches. See the manual for the full specifications of these files, or take a look in the resources folder to see how the existing ones work. If you're not comfortable doing this, send an email to support@cadenzasoftware.com and we'll be happy to do it for you. If you write your own and it works, send it to us so we can include it in future releases.
What keyboards do you recommend?
Anything that transmits MIDI (which is almost anything). If you're using multiple keyboards, it makes things easier if the keyboard has the standard 5-pin MIDI OUT port, as opposed to USB. If you're in need of a cheap solution, the M-Audio Keystation series is great.
What synthesizers do you recommend?
The Roland JV-1080 is the classic workhorse in the industry. Many musicals have been written around it, so if you're doing one of those, it's the natural choice. They can be found used for around $200.
What MIDI interfaces do you recommend?

For a 1x1 MIDI-to-USB cable, the only thing worth getting is the Roland UM-ONE. Cheaper cables are unreliable in unpredictable and disastrous ways—some will drop a MIDI message when too many occur at once. This is an annoyance when it's a NOTE ON message that's dropped, and an embarrassing disaster if it's a NOTE OFF.

If you have more than one input, then you have a few options. If 16 MIDI channels will be enough, the products from MIDI Solutions are a simple solution. They have several kinds of MIDI mergers which could not be any easier to use—they don't even require power.

Another option is to use an interface with more connections, such as the ones from MOTU.

What kinds of applications is Cadenza good for?
Any kind of live performance by MIDI-enabled instruments. Cadenza was designed with musical theater in mind, but could easily be used for anything else. So far, it has been used with upwards of 4 keyboards simultaneously, guitar, drum pads, and MalletKATs.
Why not just use Apple MainStage?
MainStage is a good solution, but it requires two expensive things: A MacBook Pro with a lot of RAM, and good sample libraries. Computers with modest amounts of RAM easily choke when trying to load samples, causing a brief gap in response or ugly audio artifacts. Cadenza can run on just about any computer and is designed to work with hardware synths or your keyboard's own built-in sounds, eliminating the sample loading problem.
Why not just use Max or Pure Data?
They offer tons of power, but are difficult to use for a musician that's not also a programmer. Cadenza offers all of the most commonly used features and is much simpler to operate.
